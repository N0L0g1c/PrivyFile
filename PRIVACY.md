# PrivyFile Privacy Policy

PrivyFile is built by VassDev Studio with a privacy-first design.

## Data Collection

**PrivyFile collects zero telemetry by default.**

- No analytics
- No crash reporting to third parties
- No file uploads
- No cloud processing

## Local Processing

All metadata reading, cleaning, and shredding happens entirely on your device. Files never leave your computer through PrivyFile.

## Local Storage

PrivyFile may store the following locally on your device:

- **Settings** — output folder, default shred method, cleaning profile
- **History** (optional) — list of files you cleaned, with privacy scores

You can disable history in Settings or clear it at any time.

## Network Access

PrivyFile does not require network access for core functionality. The only optional network use is:

- Opening external links (e.g. VassDev website) when you click them
- Future optional update checks (user-controlled, not enabled by default)

## Third-Party Tools

PrivyFile may bundle [ExifTool](https://exiftool.org/) as an external process for metadata handling. ExifTool runs locally and does not transmit your files.

## Open Source

Full source code is available at [github.com/N0L0g1c/PrivyFile](https://github.com/N0L0g1c/PrivyFile) for transparency and audit.

## Contact

Report privacy concerns via GitHub Issues on the repository.
